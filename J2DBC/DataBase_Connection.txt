There are 5 steps to the process:

1. register driver -> `Class.forName("com.mysql.jdbc.Driver");`
2. create connection -> `Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/emp","root","root");`
3. create statement -> `Statement stmt = con.createStatement();`
4. execute queries -> `ResultSet rs = stmt.executeQuery("select * from emp");`
5. close connection -> `con.close();`
